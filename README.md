## Objective
This project explores a simple quantitative breakout strategy on EUR/USD during the London session.
The goal is not profitability, but to develop quantitative trading skills:
- rule-based strategies
- backtesting
- statistical analysis
- risk awareness

## Strategy Logic
- Market: EUR/USD
- Timeframe: M5
- Session: London
- Entry:
  - Buy when price breaks the Asian session high
  - Sell when price breaks the Asian session low
- Risk Management:
  - Fixed stop-loss
  - Fixed take-profit
- Max one trade per day

## Why This Project
This strategy is intentionally simple.
It serves as a learning project to understand:
- whether session breakouts have statistical edge
- how volatility impacts results
- limitations of naive breakout strategies

## Disclaimer
This is an educational project.
It is not financial advice and not intended for live trading.
